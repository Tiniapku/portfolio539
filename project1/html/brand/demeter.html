气味图书馆/Demeter

DEMETER香水是美国纽约的品牌，创立人是美国的两个大男生,他们拥有一座收集了800多种香味的香味图书馆，立志要以"串连香味与记忆"的方式来调制香水。两人的创作灵感来自生活，帮助别人找回遗忘的嗅觉记忆。气味图书馆是DEMETER香水在中国内地的总代理。其实叫"气味图书馆"那是有来历的，这个香水的品牌"Demeter"是由两个好玩的大男生Christopher Brosius和Christopher Gable，他们拥有一座收集了800多种香味的香味图书馆，立志要以"串连香味与记忆"的方式来调制香水。 Brosius喜欢把指甲涂成蓝色，Gable则高唱"把香水民主化"的论调，他们设计出来的香水常让人跌破眼镜、匪夷所思。来自美国的DEMETER要颠覆你对香水的看法，他们的创作灵感来自于生活之中，它能帮你找回遗忘已久的嗅觉记忆。DEMETER在2000年荣获香水大奖FiFi Awards颁发最具创意香氛。

Demeter香氛于1994 年正式在美国发售，由 Christopher Brosius 和 Christopher Gable 合力创办。 Christopher Brosius 经过多年在化妆品界不断致力研究新香水味而创制出 Demeter 香氛，共 9 个系列 — Nature 、 Flower 、 Sweets 、 Vegetable 、 Fruits 、 Drink (Tea &Coffee) 、 Drink (Juice) 、 Drink (Alcohol) 和 Life 。

2000 、 2001 年， Demeter 的 "Pick-Me-Up-Snow" 荣获国际香水组织 The Fragrance Foundation 颁发 "U．S． Women′s Fragrance Star of the Year" 及 "U．S． Men′s Fragrance Star of the Year" ，这两项殊荣进一步印证了 DEMETER 的魅力。 Demeter 的 "Snow" 花了超过 5 年时间研制而成的。制作人忆起童年时在宾夕法尼亚洲乡村地方的纯白新雪的香味，他那把雪粒放进口中的那种回忆，透过 Snow 香水重现。 独特的 Demeter 在美国各地极受欢迎，连 Sharon Stone ( 莎朗斯通) 、 Kate Moss ( 凯特摩丝) 、 Cher ( 雪儿 ) 、 Drew Barrymore ( 珠儿巴莉摩 ) 、 Courteney Cox Arquette ( 科特尼 - 考克斯 - 阿奎特 ) 、 Uma Thurman ( 乌玛舒曼) 、 Stella McCartney 、名模 Tyra Banks 、 Clint Eastwood ( 克林伊斯威特 ) 和 Candice Bergen ( 肯蒂丝柏根 ) 等多位欧美红星和名模都深受其香气吸引。 DEMETER 的出现也席卷邻近的亚洲日本市场，不但改变了日本女性对香水的一贯概念，同时也获得了日本男性的认同。 Snow( 雪 ) 、 Rain( 雨 ) 、 Gingerale( 姜汁汽水 ) 、 Laundromat( 洗衣间 ) 及 Peach( 蜜桃 ) 这 5 款香氛 是日本杂志追捧的潮流人气产品。另外像是华人巨星张曼玉也特别钟情于Gin &Tonic ( 金汤尼 ) 酒香微醺带出有点暗示有点放电的诱人香氛。
