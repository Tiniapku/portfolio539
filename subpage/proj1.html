PerfumeDaily

PerfumeDaily (see here) is a course project of Web architecture supervised by professor Wen Zhao. This website was made in 2014 together with Dan Lou and Zhenyu Ding, when we were all junior undergraduate students in Peking University

The intended audience of this website are perfume lovers. We want provide useful informatioin for people to choose perfume for themselves and for gifts. Famous perfume brands are also introduced

We worked on this project for about two weeks, and tried us best to collect as much information as we could to build a integrated and inter-connected information. However because we were all novice to website development, we didn't finish all. Also we needed to do more to implement it as a dynamic website for better usage.

It is still a mystery for me to keep all the webpages' style consistent. In our process, we firstly designed our index.html's layout and built up the css document, then we copy and paste the codes that define shared layout and style into every page. But it is rather difficult for me to edit it if I want the change applies to every page.
